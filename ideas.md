## 方案一：极简主义商务风 (Minimalist Business)

**设计运动**: 瑞士国际主义风格 (Swiss Style) / 现代极简主义 (Modern Minimalism)

**核心原则**:
1.  **内容至上**: 去除一切不必要的装饰，让 Listing 图片和关键数据成为视觉焦点。
2.  **网格系统**: 严格的数学网格布局，传达专业、严谨和可信赖感。
3.  **清晰层级**: 通过字体大小和粗细的强烈对比来构建信息层级，而非颜色。
4.  **功能美学**: 每一个元素都必须有其功能性，形式追随功能。

**色彩哲学**:
*   **主色**: 商务深蓝 (#002349) - 象征专业、稳重、信任，符合商业地产属性。
*   **背景**: 纯白 (#FFFFFF) 与 极浅灰 (#F5F5F7) - 模仿 Apple 官网的干净背景，最大化留白。
*   **强调色**: 雅致金 (#C5A065) - 仅用于极少量的CTA按钮或高亮数据，增添一丝尊贵感。
*   **情感意图**: 传达一种“所见即所得”的透明感和高端服务的专业度。

**布局范式**:
*   **非对称平衡**: 首页 Banner 采用左文右图或非对称分割，打破传统对称的沉闷。
*   **卡片式列表**: Listing 展示采用悬浮感极弱的卡片，强调图片本身的质感。
*   **大留白**: 模块之间保持极大的垂直间距（120px+），让阅读呼吸感极强。

**标志性元素**:
1.  **细线条**: 使用 1px 的极细分割线，精致且不抢眼。
2.  **大字号标题**: 页面标题使用超大字号（4rem+），形成强烈的视觉冲击。
3.  **高质量摄影**: 图片采用 16:9 或 3:2 的经典比例，无圆角或微圆角（4px）。

**交互哲学**:
*   **微交互**: 按钮悬停时的细微背景色变化或位移，而非剧烈缩放。
*   **平滑滚动**: 页面滚动时的视差效果极简，仅保留图片懒加载的淡入效果。

**动画**:
*   **淡入淡出**: 元素进入视口时采用缓慢的 Opacity 变化 (0 -> 1)，时长 0.8s，Ease-out。
*   **无缝轮播**: Listing 图片轮播采用无缝切换，无多余特效。

**排版系统**:
*   **标题**: SF Pro Display / PingFang SC (Bold/Semibold) - 现代无衬线体，简洁有力。
*   **正文**: SF Pro Text / PingFang SC (Regular) - 易读性极高。
*   **数字**: DIN Alternate 或类似工业风格字体，强调数据的专业性。

## 方案二：沉浸式叙事风 (Immersive Narrative)

**设计运动**: 杂志排版风格 (Editorial Design)

**核心原则**:
1.  **故事驱动**: 将每一个 Listing 视为一个独立的故事，通过排版引导阅读流。
2.  **视觉冲击**: 使用全屏或大幅图片作为背景，创造身临其境的看房体验。
3.  **情感连接**: 强调“家”与“事业”的情感联系，而非冷冰冰的数据。
4.  **打破常规**: 布局上打破传统的 Grid，使用错位和重叠。

**色彩哲学**:
*   **主色**: 商务深蓝 (#002349) - 作为文字和遮罩层颜色。
*   **背景**: 米白 (#FAFAF5) - 模拟纸张质感，比纯白更温润。
*   **辅助色**: 炭灰 (#333333) - 用于正文，比纯黑更柔和。
*   **情感意图**: 营造一种阅读高端地产杂志的优雅体验，建立与客户的情感共鸣。

**布局范式**:
*   **杂志布局**: 图片与文字的错位排版，文字可能覆盖在图片边缘。
*   **全屏滚动**: 首页采用全屏滚动 (Scroll Snap)，每一屏展示一个核心卖点或精选 Listing。

**标志性元素**:
1.  **衬线字体**: 标题使用优雅的衬线体，增加人文气息。
2.  **首字下沉**: 文章或描述段落使用首字下沉设计。
3.  **重叠图层**: 图片与色块、文字之间的微妙重叠。

**交互哲学**:
*   **滚动触发**: 随着滚动，文字和图片有节奏地出现，如同翻阅杂志。
*   **光标跟随**: 自定义光标，悬停在图片上时变为“查看详情”提示。

**动画**:
*   **图片视差**: 背景图片滚动速度慢于前景文字，产生深度感。
*   **文字揭示**: 标题文字逐行或逐词滑入。

**排版系统**:
*   **标题**: Noto Serif SC / Songti SC - 优雅的宋体/衬线体。
*   **正文**: PingFang SC (Light) - 纤细的无衬线体，保持现代感。

## 方案三：科技未来风 (Tech-Forward Future)

**设计运动**: 玻璃拟态 (Glassmorphism) / 科技现代主义

**核心原则**:
1.  **数据可视化**: 强调数据的图形化展示，如投资回报率图表、地图热点。
2.  **通透感**: 使用半透明磨砂玻璃效果，增加界面的层次感和现代感。
3.  **动态响应**: 界面对用户的操作有即时且丰富的反馈。
4.  **智能感**: 搜索和筛选功能作为核心交互入口，置于显眼位置。

**色彩哲学**:
*   **主色**: 商务深蓝 (#002349) - 作为深色模式的基底或浅色模式的强调。
*   **背景**: 极浅蓝灰 (#F0F4F8) - 带有科技感的冷色调背景。
*   **强调色**: 电光蓝 (#007AFF) - 用于交互元素，致敬 iOS 系统色。
*   **情感意图**: 传达高效、智能、前沿的商业地产服务形象。

**布局范式**:
*   **仪表盘式**: 首页首屏即展示核心筛选器和地图概览。
*   **悬浮层**: 内容板块以悬浮卡片形式存在于背景之上。

**标志性元素**:
1.  **磨砂玻璃**: 导航栏、卡片背景使用 Backdrop Filter Blur。
2.  **圆角矩形**: 统一的大圆角 (16px-24px)，亲和且现代。
3.  **渐变光晕**: 背景中微妙的蓝色光晕，增加氛围感。

**交互哲学**:
*   **即时筛选**: 筛选条件改变时，列表无刷新即时更新（React State）。
*   **深度反馈**: 点击按钮时的涟漪效果 (Ripple)。

**动画**:
*   **弹性动画**: 元素出现和移动带有弹性 (Spring Animation)。
*   **数字滚动**: 价格和面积数字从 0 滚动到最终值。

**排版系统**:
*   **标题**: Roboto / Helvetica Neue - 经典的现代无衬线体。
*   **正文**: System Font - 使用系统默认字体，确保最快的渲染和熟悉感。
